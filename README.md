# University Lost and Found Portal

A web-based portal that allows university students to report lost items and help others locate them.

## Features
- Submit a lost item report through a form
- Display submitted items in a dynamic list
- Contact information included for item owners
- JavaScript event listeners used to process form submissions

## Technologies
HTML
CSS
JavaScript

## How It Works
Users fill out a form with item details such as category, location, and contact information.  
When the form is submitted, JavaScript stores the data in an array and dynamically updates the item listing on the page.
